Game Project / Back End
=============
Project 'ADI' Socket Manage Server(Master) . 

### How To Run? ###
- dependency install
```sh
$ npm install
```
- build and run as develop mode
```sh
$ npm start
```
- build and run as production mode
```sh
$ npm run build
```

### Server ###
- AWS EC2 Instance
- PM2
- Node.js
- Express

### Back Dev Environments ###
- Typescript
- Javascript
- MySQL
- Webpack 4
